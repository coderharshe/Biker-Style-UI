Build a frontend-only mobile-first web app for a biker community and safety platform.

Tech Requirements:

Use React + Tailwind CSS

Dark theme default

Modern glassmorphism + neon accent UI

No backend

Use mock data and local state

Mobile responsive (primary target mobile screen)

Primary Accent: Neon Orange

Secondary Accent: Electric Blue

SOS Color: Red (#FF3B30)

Text: White / Light Grey

Navigation:

Home | Groups | Tasks | Leaderboard | Profile
